## ADS
Private는 내가 아는 사람들이 방문했던, 믿을 만한 장소들만 보여준다. 왜냐하면, 너무 많은 광고로 인해 어디를 가야할지 모르는 사람들이 많기 때문이다. 내가 팔로우한 사람들이 방문한 장소들만 볼 수 있기 때문에 어디를 갈지 쉽게 정할 수 있다. 해당 장소에 방문했던 사람들에게 빠르게 채팅으로 후기를 물어볼 수 있다.

## 키워드
- iOS 16.4+
- MVVM
- 다크 모드 대응
